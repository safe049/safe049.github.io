# 序
我的Arch可真是越来越乱了，故，我决定重装系统，不过这次我打算尝点新的，就去装了Fedora

最开始我想装OpenSUSE，但是...

如下

```
安装：5/5，高自定义度的同时点点就行，速度快
安装后处理：2，grub双系统需要复杂调试，键盘和鼠标等usb设备经常需要很久才能被加载好
使用：
输入法：自带中文拼音输入法，体验拉满
大部分国内软件包括qq需要使用flatpak安装
包管理器强大
yast不太符合我个人口味，但确实强大
3/5

我不行了，我要去用fedora了，opensuse也许很好，但他似乎和我的计算机很不兼容
```
我遇到一大堆乱的很的毛病，所以换Fedora了


# 体验

- 安装：好！虽然自定义程度低，但一键完成，真一键，而且自动适配了Windows加上了引导，也有自带的拼音输入法
- 使用：社区生态稍逊，相比Arch的AUR完全没面子，但论日用特别好，在国内环境下大部分软件有RPM，方便。

总的来说，安装打5分，使用4分

后面的我还没怎么体验到，，另外，包管理器很强大。
